# Sistema_Compras_REFATORADO_TRYCATCH_ENUMS_HERAN-AS-
Sistema de compras refatorado para adoção de boas práticas de POO e robustez de código. Implementação de herança para especialização de produtos/clientes, enums para controle estrito de status e tipos, e tratamento de exceções com blocos try-catch para garantir resiliência e evitar falhas críticas durante o fluxo de execução.
